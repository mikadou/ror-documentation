enum scriptEvents

This enum describes what events are existing. The script can register to receive events.

Enumerator
SE_COLLISION_BOX_ENTER 

triggered when truck or person enters a previous registered collision box, the argument refers to the collision box ID

SE_COLLISION_BOX_LEAVE 

triggered when truck or person leaves a previous registered and entered collision box, the argument refers to the collision box ID

SE_TRUCK_ENTER 

triggered when switching from person mode to truck mode, the argument refers to the truck number

SE_TRUCK_EXIT 

triggered when switching from truck mode to person mode, the argument refers to the truck number

SE_TRUCK_ENGINE_DIED 

triggered when the trucks engine dies (from underrev, water, etc), the argument refers to the truck number

SE_TRUCK_ENGINE_FIRE 

triggered when the planes engines start to get on fire, the argument refers to the truck number

SE_TRUCK_TOUCHED_WATER 

triggered when any part of the truck touches water, the argument refers to the truck number

SE_TRUCK_BEAM_BROKE 

triggered when a beam breaks, the argument refers to the truck number

SE_TRUCK_LOCKED 

triggered when the truck got lock to another truck, the argument refers to the truck number

SE_TRUCK_UNLOCKED 

triggered when the truck unlocks again, the argument refers to the truck number

SE_TRUCK_LIGHT_TOGGLE 

triggered when the main light is toggled, the argument refers to the truck number

SE_TRUCK_SKELETON_TOGGLE

SE_GENERIC_NEW_TRUCK

SE_GENERIC_DELETED_TRUCK

SE_GENERIC_INPUT_EVENT

SE_GENERIC_MOUSE_BEAM_INTERACTION

SE_TRUCK_TRACTIONCONTROL_TOGGLE

SE_TRUCK_ANTILOCKBRAKE_TOGGLE
