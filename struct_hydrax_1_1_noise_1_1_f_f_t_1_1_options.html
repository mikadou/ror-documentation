Ogre::Vector3 Hydrax::Noise::FFT::Options::GPU_LODParameters

LOD Parameters, in order to obtain a smooth normal map we need to decrease the detail level when the pixel is far to the camera. This parameters are stored in an Ogre::Vector3: x -> Initial LOD value (Bigger values -> less detail) y -> Final LOD value z -> Final distance
